# Kidanemaryam Wagaw Reta Academic Website

This repository contains the source files for the academic website of **Kidanemaryam Wagaw Reta**.

Live website:

https://rekidane.github.io/kidane/

## About

Kidanemaryam Wagaw Reta is a plant physiologist and plant biochemist working on plant stress physiology, metabolomics, ionomics, multi-omics integration, predictive modeling, and systems crop science.

The website presents research interests, publications, conference presentations, projects, teaching and mentoring, people and collaborators, gallery items, contact information, and a downloadable CV.

## Main Content Files

- `_pages/about.md` controls the homepage and biography.
- `_pages/research.md` controls the research overview.
- `_pages/publications.md` controls the publications page.
- `_pages/presentations.md` controls conference presentations.
- `_pages/projects.md` controls the projects page.
- `_projects/` contains individual project pages.
- `_pages/teaching.md` controls teaching and mentoring.
- `_pages/service.md` controls service and professional activities.
- `_pages/gallery.md` controls the gallery.
- `_pages/profiles.md` controls People and Collaborators.
- `_pages/contact.md` controls contact information and the map.
- `_pages/cv.md` controls the CV page.
- `_data/cv.yml` stores structured CV information.
- `_bibliography/papers.bib` stores BibTeX publication entries.
- `_config.yml` controls site-wide settings.
